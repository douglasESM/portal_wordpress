== Description ==

Displays a simple widget with a list of the most commented posts.

* English and German Translation
* Number of Posts choosable
* Multiple widgets possible
* More Information will be added as the plugin gets better and bigger.

Features in the future

* Choose year or period of time
* Choose different posts types (Posts, Pages, ...)

== Installation ==


1. Upload `pd-mostcommentedposts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag & Drop the Widget in the Widget Area
